# 🚀 ReviewAI – eCommerce Review Analytics Dashboard

ReviewAI is a modern, AI-powered dashboard that helps brands analyze customer reviews across multiple platforms like Amazon, Flipkart, and Shopify.

It provides real-time insights, sentiment tracking, smart alerts, and automated report generation to help businesses make better decisions.

---

## ✨ Features

### 📊 Dashboard Overview

* Real-time KPI tracking (reviews, ratings, sentiment)
* Beautiful animated stat cards
* Multi-platform analytics (Amazon, Flipkart, Shopify)

### 📈 Data Visualization

* Volume trends chart
* Rating distribution chart
* Interactive UI with smooth animations (Framer Motion)

### 🔍 Smart Filtering

* Filter reviews by platform
* Filter by rating (1★ to 5★)
* Date-based filtering (Last 30 days)

### 🤖 AI Insights

* AI-generated review summaries
* Sentiment trend analysis
* Actionable recommendations

### 🚨 Smart Alerts

* Detect critical negative reviews
* Highlight important issues instantly

### 📄 PDF Report Generation

* Generate **Executive Reports**
* Includes:

  * Key metrics table
  * Trend indicators
  * AI insights summary
* Fully branded and professional layout

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/bruntha24/Review_AI.git
cd Review_AI
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

```bash
npm run dev
```
